Draw the graph of the In equation 3x + 2y ≤ 6.

To graph the inequality 3x + 2y ≤ 6, follow these steps:

  1. First, rewrite the inequality in slope-intercept form (y = mx + b): 2y ≤ -3x + 6, then y ≤ -3/2x + 3.
  2. Plot the boundary line y = -3/2x + 3 as a dashed line since the inequality is not an equality.
  3. Choose a test point that does not lie on the boundary line, such as (0, 0).
  4. Substitute the test point into the inequality: 0 ≤ -3/2(0) + 3, which is true.
  5. Since the test point satisfies the inequality, shade the region containing the test point.
