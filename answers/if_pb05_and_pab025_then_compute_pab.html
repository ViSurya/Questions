If P(B)=0.5 and P(A∩B)=0.25, then compute P(A/B).

Given that P(B) = 0.5 and P(A ∩ B) = 0.25, we can calculate P(A/B) using the conditional probability formula: P(A/B) = P(A ∩ B) / P(B).

Substituting the given values, we get: P(A/B) = 0.25 / 0.5 = 0.5.

Therefore, the probability of event A occurring given that event B has already occurred is 0.5.
