Verify that the function y= a cosx + b sinx, where a, b ∈ R is a solution of the differential equation d^2y / dx^2 + y = 0

To verify that \(y = a \cos x + b \sin x\), where \(a, b \in \mathbb{R}\), is a solution to the differential equation \(\frac{d^2y}{dx^2} + y = 0\), we can substitute \(y\) and its derivatives into the equation and check if it holds true.

Differentiating \(y\) with respect to \(x\), we get \(\frac{dy}{dx} = -a \sin x + b \cos x\). Differentiating again, we get \(\frac{d^2y}{dx^2} = -a \cos x - b \sin x\).

Substituting \(y\), \(\frac{dy}{dx}\), and \(\frac{d^2y}{dx^2}\) into the differential equation, we get:

\( (-a \cos x - b \sin x) + (a \cos x + b \sin x) = 0 \)

Simplifying the equation, we get \(0 = 0\), which is true. Therefore, \(y = a \cos x + b \sin x\) is a solution to the differential equation \(\frac{d^2y}{dx^2} + y = 0\).
