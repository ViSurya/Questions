If A = [2 4, 3 2] and B = [1 3, 2 5], then find the value of A.B.

To find the value of A.B, we need to multiply the elements of the first row of matrix A with the corresponding elements of the first column of matrix B, and then add the products. We repeat this process for the second row of matrix A and the second column of matrix B. Finally, we place the results in a new matrix.

A.B [2 * 1 + 4 * 2, 2 * 3 + 4 * 5]
= [2 + 8, 6 + 20]
= [10, 26]
