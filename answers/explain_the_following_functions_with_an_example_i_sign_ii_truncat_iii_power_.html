Explain the following functions with an example : (i) SIGN ( ) (ii) TRUNCAT ( ) (iii) POWER ( )

```html

SIGN ( ): The SIGN function returns the sign of a number. The syntax is SIGN(number). For example, SIGN(5) returns 1, SIGN(-5) returns -1, and SIGN(0) returns 0.

TRUNCAT ( ): The TRUNCAT function truncates a number to the specified number of decimal places. The syntax is TRUNCAT(number, decimal_places). For example, TRUNCAT(3.14159, 2) returns 3.14, and TRUNCAT(-4.56789, 0) returns -5.

POWER ( ): The POWER function raises a number to the specified power. The syntax is POWER(number, power). For example, POWER(2, 3) returns 8, and POWER(5, -2) returns 0.2.
```
