Find the value of ∫(dx)/(5+4sinx)

To find the value of the integral ∫(dx)/(5+4sinx), we can use the substitution method. Let u = 5 + 4sinx. Then, du/dx = 4cosx, and dx = du/(4cosx). Substituting these into the integral, we get:

∫(dx)/(5+4sinx) = ∫(1/u)(du/(4cosx))
= (1/4)∫(1/u)du = (1/4)ln|u| + C
= (1/4)ln|5+4sinx| + C where C is the constant of integration.
