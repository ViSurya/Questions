Find the value of dy/dx, if x = a (t + sin t), y = a (1 – cos t)

To find the value of dy/dx, we can use the chain rule. Differentiating both x and y with respect to t, we get:

dx/dt = a(1 + cos t)

...

(1)

dy/dt = a sin t

...

(2)

Dividing (2) by (1), we get:

dy/dx = (a sin t) / (a(1 + cos t))

...

(3)

Simplifying (3), we get the final result:

dy/dx = tan(t/2)
