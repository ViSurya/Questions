If f(x) = x^2 and g(r) = x + 3. X ∈ R, then find the value of (gof)x, (fog)x, (fog)2.

Given functions: f(x) = x^2 and g(r) = x + 3. X ∈ R.

To find: (gof)x, (fog)x, (fog)2.

Solution:

  1. (gof)x = g(f(x)) = g(x^2) = (x^2) + 3 = x^2 + 3.
  2. (fog)x = f(g(x)) = f(x + 3) = (x + 3)^2 = x^2 + 6x + 9.
  3. (fog)2 = f(g(2)) = f(2 + 3) = f(5) = 5^2 = 25.
