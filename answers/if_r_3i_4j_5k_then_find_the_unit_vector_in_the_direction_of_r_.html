If r ⃗=3i ⃗-4j ⃗+5k ⃗ then find the unit vector in the direction of r ⃗

To find the unit vector in the direction of r ⃗, we first need to calculate its magnitude. The magnitude of r ⃗ is given by the formula: $$|r ⃗| = sqrt((3)^2 + (-4)^2 + (5)^2) = sqrt(9 + 16 + 25) = sqrt(50) = 5sqrt(2)$$

Therefore, the unit vector in the direction of r ⃗ is given by: $$hat{r} = r ⃗ / |r ⃗| = (3i ⃗-4j ⃗+5k ⃗) / 5sqrt(2) = (3/5sqrt(2))i ⃗-(4/5sqrt(2))j ⃗+(5/5sqrt(2))k ⃗$$
