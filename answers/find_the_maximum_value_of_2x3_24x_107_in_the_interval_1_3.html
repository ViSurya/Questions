Find the maximum value of 2x3 - 24x + 107 in the interval [1, 3].

To find the maximum value of the function \(f(x) = 2x^3 - 24x + 107\) in the interval [1, 3], we can use the first derivative test.

The first derivative of \(f(x)\) is \(f'(x) = 6x^2 - 24\). Setting \(f'(x) = 0\), we get \(x = 2\).

Since \(f'(x)\) is positive for \(x < 2\) and negative for \(x > 2\), \(x = 2\) is a point of maximum.

Evaluating \(f(x)\) at \(x = 2\), we get the maximum value of \(f(x)\) in the interval [1, 3] as \(f(2) = 111\).
