Find the unit vector in the direction of vector

To find the unit vector in the direction of vector a, we first calculate the magnitude of a using the formula |a| = √(ax2 + ay2 + az2). Then, we divide each component of a by its magnitude to obtain the unit vector û.

Mathematically, the unit vector in the direction of a is given by:

û = (ax / |a|, ay / |a|, az / |a|)
