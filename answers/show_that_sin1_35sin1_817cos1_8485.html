Show that sin^(-1) 3/5-sin^(-1) 8/17=cos^(-1) 84/85

To prove the identity sin^(-1) 3/5 - sin^(-1) 8/17 = cos^(-1) 84/85, we can utilize the identity sin(A - B) = sinA cosB - cosA sinB. Let A = sin^(-1) 3/5 and B = sin^(-1) 8/17. Then, sin(A - B) = sin(sin^(-1) 3/5 - sin^(-1) 8/17) = (3/5)(17/sqrt(17^2 - 8^2)) - (8/17)(sqrt(5^2 - 3^2)/5) = (51/85) - (24/85) = 27/85. Now, we can use the identity cos^(-1) x = pi/2 - sin^(-1) x to find cos^(-1) 84/85 = pi/2 - sin^(-1) 84/85. Substituting the value of sin(A - B), we get cos^(-1) 84/85 = pi/2 - sin^(-1) 27/85. Therefore, sin^(-1) 3/5 - sin^(-1) 8/17 = cos^(-1) 84/85.
