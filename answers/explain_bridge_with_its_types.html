Explain bridge with its types.

A bridge is a network device that connects two or more networks together. It operates at the data link layer of the OSI model and forwards data packets between networks. There are various types of bridges, including:

  1. Transparent Bridge: Operates silently in the background, learning the MAC addresses of devices on each network and forwarding data packets accordingly.
  2. Source Route Bridge: Uses a routing table to determine the path that data packets should take, allowing for more efficient routing.
  3. Translational Bridge: Converts data packets from one protocol to another, enabling communication between networks using different protocols.
