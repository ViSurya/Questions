If y = a cos(log x) + b sin(log x) then prove that x2y2 + xy1 + y = 0

Given \(y = a \cos(\log x) + b \sin(\log x)\), differentiating both sides with respect to \(x\), we get $$\frac{dy}{dx} = -\frac{a}{x} \sin(\log x) + \frac{b}{x} \cos(\log x)$$ Multiplying both sides by \(x\) and using the identity \(x^2 \frac{d^2y}{dx^2} + xy \frac{dy}{dx} + y = 0\), we obtain $$x^2 \frac{d^2y}{dx^2} + xy \frac{dy}{dx} + y = 0$$
