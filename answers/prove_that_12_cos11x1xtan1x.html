Prove that : 1/2 cos^1[(1-x)/(1+x)=tan^(-1)⁡[√x]]

To prove the identity, we can start by using the identity \(tan^2 x = sec^2 x - 1\). Substituting \(x = tan^{-1} [\sqrt{x}]\), we get:

$$tan^2 (tan^{-1} [\sqrt{x}]) = sec^2 (tan^{-1} [\sqrt{x}]) - 1$$

Simplifying the left-hand side using the identity \(tan^2 x = \frac{sin^2 x}{cos^2 x}\), we get:

$$\frac{sin^2 (tan^{-1} [\sqrt{x}])}{cos^2 (tan^{-1} [\sqrt{x}])} = sec^2 (tan^{-1} [\sqrt{x}]) - 1$$

Substituting \(sec^2 x = 1 + tan^2 x\), we get:

$$\frac{sin^2 (tan^{-1} [\sqrt{x}])}{cos^2 (tan^{-1} [\sqrt{x}])} = 1 + tan^2 (tan^{-1} [\sqrt{x}]) - 1$$

Simplifying further, we get:

$$\frac{sin^2 (tan^{-1} [\sqrt{x}])}{cos^2 (tan^{-1} [\sqrt{x}])} = tan^2 (tan^{-1} [\sqrt{x}])$$

Taking the square root of both sides, we get:

$$\frac{sin (tan^{-1} [\sqrt{x}])}{cos (tan^{-1} [\sqrt{x}])} = tan (tan^{-1} [\sqrt{x}])$$

Finally, using the identity \(tan (tan^{-1} x) = x\), we get:

$$\frac{1}{2} cos^{-1} \left[\frac{1-x}{1+x}\right] = tan^{-1} [\sqrt{x}]$$

Hence, the identity is proved.
