Prove that ∫sec x dx = log (sec x + tan x) + c

To prove the integral of sec x with respect to x is equal to the natural logarithm of sec x plus tan x plus a constant, we can use the substitution method. Let u = sec x + tan x. Then, du/dx = sec x tan x + sec^2 x = sec x (sec x + tan x). Substituting into the integral, we get:

∫ sec x dx = ∫ (1/u) du
= ln |u| + c = ln |sec x + tan x| + c
