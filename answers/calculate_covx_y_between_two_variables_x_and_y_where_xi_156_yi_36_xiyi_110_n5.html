Calculate cov(X, Y) between two variables x and y where : ∑xi = 156, ∑yi = 36, ∑xiyi =110, n=5

The covariance between two variables x and y, denoted as cov(X, Y), is a measure of their linear relationship. It is calculated using the formula:

cov(X, Y) = 1/n * ∑(xi - x̄)(yi - ȳ)

where x̄ and ȳ are the means of x and y, respectively, and n is the number of data points.

In this case, we have ∑xi = 156, ∑yi = 36, ∑xiyi = 110, and n = 5. Plugging these values into the formula, we get:

cov(X, Y) = 1/5 * (156 - 156/5)(36 - 36/5) = 0

Therefore, the covariance between x and y is 0, indicating that there is no linear relationship between the two variables.
