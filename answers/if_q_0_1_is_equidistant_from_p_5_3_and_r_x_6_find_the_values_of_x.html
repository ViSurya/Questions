If Q (0, 1) is equidistant from P (5, -3) and R (x, 6). Find the values of x.

Since Q is equidistant from P and R, the distance between Q and P is equal to the distance between Q and R. Using the distance formula, we can set up an equation and solve for x.

Distance between Q and P Distance between Q and R
$$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$ $$d = \sqrt{(5 - 0)^2 + (-3 - 1)^2}$$ $$d = \sqrt{25 + 16}$$ $$d = \sqrt{41}$$ $$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$ $$d = \sqrt{(x - 0)^2 + (6 - 1)^2}$$ $$d = \sqrt{x^2 + 25}$$

Setting the two distances equal to each other, we get:

$$\sqrt{41} = \sqrt{x^2 + 25}$$ $$41 = x^2 + 25$$ $$x^2 = 16$$ $$x = \pm 4$$

Therefore, the values of x are 4 and -4.
