Verify Rolle's theorem for function f(x)=x2-1 in the interval [–1, 1).

Rolle's theorem states that if a function f(x) is continuous on the closed interval [a, b], differentiable on the open interval (a, b), and f(a) = f(b), then there exists at least one number c in (a, b) such that f'(c) = 0.

For the function f(x) = x2 - 1 in the interval [-1, 1), we have:

  1. f(x) is continuous on the closed interval [-1, 1] because it is a polynomial function.
  2. f(x) is differentiable on the open interval (-1, 1) because it is a polynomial function.
  3. f(-1) = (-1)2 - 1 = 0 and f(1) = (1)2 - 1 = 0, so f(-1) = f(1).

Therefore, by Rolle's theorem, there exists at least one number c in (-1, 1) such that f'(c) = 0.
