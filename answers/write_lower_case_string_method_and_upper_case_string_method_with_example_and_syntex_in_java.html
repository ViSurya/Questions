Write lower case string method and upper case string method with example and syntex in Java.

The `toLowerCase()` method converts a string to lowercase, while the `toUpperCase()` method converts a string to uppercase.

Method Syntax Example
`toLowerCase()` `string.toLowerCase()` `"HELLO".toLowerCase() // "hello"`
`toUpperCase()` `string.toUpperCase()` `"hello".toUpperCase() // "HELLO"`
