Write down salient features of genetic code.

The genetic code is a set of rules that determines how information encoded in genetic material (DNA or RNA) is translated into proteins. It is a universal code, meaning that it is the same for all living organisms. The genetic code is composed of codons, which are sequences of three nucleotides that specify a particular amino acid or a stop signal. There are 64 possible codons, which code for 20 amino acids and three stop signals. The genetic code is read in a specific direction, and the sequence of codons determines the sequence of amino acids in a protein.
