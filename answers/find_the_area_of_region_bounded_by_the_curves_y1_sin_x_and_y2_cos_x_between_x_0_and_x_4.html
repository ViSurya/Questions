Find the area of region bounded by the curves y1 = sin x and y2 = cos x between x = 0 and x = Π /4.

The area of the region bounded by the curves y1 = sin x and y2 = cos x between x = 0 and x = Π /4 can be calculated using integration.

The area is given by the integral of the difference between the two functions with respect to x, evaluated from x = 0 to x = Π /4.

Therefore, the area is: $$Area = \int_0^{\frac{\pi}{4}} (\sin x - \cos x) \ dx$$
