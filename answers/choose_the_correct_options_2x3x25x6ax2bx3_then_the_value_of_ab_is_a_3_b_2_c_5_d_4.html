Choose the correct options : (2x+3)/(x^2+5x+6)=A/(x+2)+B/(x+3), then the value of A+B is : (a) 3 (b) 2 (c) 5 (d) 4

To find the value of A+B, we can compare the coefficients of x and the constant terms on both sides of the equation. By equating the coefficients, we get A = 1 and B = 2. Therefore, A+B = 1 + 2 = 3.

Hence, the correct option is (a) 3.
